NetSentry inference API is deploying.

1. Watch the rollout:
   kubectl -n {{ .Release.Namespace }} rollout status deploy/{{ include "netsentry.fullname" . }}

2. Port-forward and hit the API:
   kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "netsentry.fullname" . }} {{ .Values.service.port }}:{{ .Values.service.port }}
   curl localhost:{{ .Values.service.port }}/health
   curl -X POST localhost:{{ .Values.service.port }}/predict \
     -H 'content-type: application/json' -d @examples/sample_flow.json
